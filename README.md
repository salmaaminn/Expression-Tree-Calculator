Run make in the Expression-Tree-Calculator directory. This will compile heap.c, preorder.c, postorder.c, inorder.c, and calculate.c into object files inside the obj directory, and then link them into executables in the bin directory.

Then run:
#./bin/preorder "expression"
#./bin/postorder "expression" 
#./bin/inorder "expression"
#./bin/calculate "expression"
#./bin/heap
